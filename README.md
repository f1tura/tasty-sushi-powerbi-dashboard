# Tasty Sushi — дашборд продаж в Power BI

Отчёт в **Power BI** по датасету продаж ресторана суши.  
Цель — показать ключевые метрики (выручка/заказы/средний чек), динамику по времени и вклад городов/категорий.

## Страницы отчёта
1. **Обзор (Overview)**
   - KPI: Выручка, Кол-во заказов, Кол-во позиций, Средний чек (AOV), Среднее кол-во позиций в заказе
   - Динамика выручки по месяцам
   - Топ категорий по выручке

2. **Города и спрос**
   - Топ городов по заказам / по выручке
   - Заказы по дням недели
   - Доля выручки по категориям (если добавлено)

3. **Анализ City × Category**
   - Матрица: **Выручка / Заказы / Средний чек (AOV)**
   - Срезы: **Год / Город / Категория**
   - Можно кликать по значениям и смотреть вклад отдельных сегментов

## Файлы в репозитории
- Отчёт Power BI: `report/Tasty_Sushi_Dashboard.pbix`
- Данные: `data/Tasty Sushi restaurant.csv`
- Скриншоты: `images/`

## Как открыть отчёт
1. Скачайте файл `.pbix` из папки `report/`
2. Откройте его в **Power BI Desktop**
3. Если Power BI попросит указать источник данных — выберите файл  
   `data/Tasty Sushi restaurant.csv`
